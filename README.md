## Projects

### Insurance Pricing Model — SUSEP AUTOSEG
End-to-end pricing model for auto insurance using real Brazilian market data (SUSEP AUTOSEG 2019–2021). Collision and theft coverages modelled separately with GLM Poisson (frequency) and GLM Gamma (severity). Compared against XGBoost Tweedie (Gini = 0.241 collision, 0.402 theft) with SHAP explainability. Interactive deploy via Streamlit.

`Python` `GLM` `XGBoost` `SHAP` `SUSEP` `Streamlit`

[github.com/arthurpmotta02/insurance-pricing-susep](https://github.com/arthurpmotta02/insurance-pricing-susep)

---

### Reinsurance Portfolio Optimization
End-to-end reinsurance analytics pipeline using French Motor TPL data (freMTPL2, 678k policies). Extreme Value Theory (GPD, Hill estimator) for tail modelling, treaty structuring (Quota Share, XL, Aggregate Stop Loss), and differential evolution optimization. Achieves 20.1% capital relief on VaR 99.5% annual aggregate. Streamlit dashboard with 5 interactive pages.

`Python` `EVT` `GPD` `Reinsurance` `Optimization` `Streamlit`

[github.com/arthurpmotta02/reinsurance-portfolio-optimization](https://github.com/arthurpmotta02/reinsurance-portfolio-optimization)

---

### Pension Fund Actuarial Analysis
Actuarial valuation of a Brazilian BD (Defined Benefit) pension plan. Lee-Carter mortality projection to 2065 via `StMoMo` (R); Projected Unit Credit valuation via `lifecontingencies` (R) — PMBaC R$16.7M, PMBC R$353.8M. Longevity risk: +1 year of life = +0.7% liability. ALM: liability duration 18.5 years vs NTN-B portfolio 9.3 years, duration gap 9.2 years. Interest rate stress test (±200bp). Streamlit dashboard.

`R` `Python` `StMoMo` `lifecontingencies` `ALM` `NTN-B` `Streamlit`

[github.com/arthurpmotta02/pension-fund-actuarial-analysis](https://github.com/arthurpmotta02/pension-fund-actuarial-analysis)
